# Success Criterion 3.1.2 - Language of Parts

On Android, you can use [LocaleSpan](https://developer.android.com/reference/android/text/style/LocaleSpan) to automatically switch between languages.

<pre>
    <code class="language-kotlin">
val locale = <a href="https://developer.android.com/reference/java/util/Locale">Locale</a>.<a href="https://developer.android.com/reference/java/util/Locale#forLanguageTag(java.lang.String)">forLanguageTag</a>("nl-NL")
val localeSpan = <a href="https://developer.android.com/reference/android/text/style/LocaleSpan">LocaleSpan</a>(locale)

val string = <a href="https://developer.android.com/reference/android/text/SpannableString">SpannableString</a>("Stichting Appt")
string.<a href="https://developer.android.com/reference/android/text/Spannable#setSpan(java.lang.Object,%20int,%20int,%20int)">setSpan</a>(localeSpan, 0, string.length-1, Spanned.SPAN_INCLUSIVE_INCLUSIVE)

element.<a href="https://developer.android.com/reference/android/widget/TextView#setText(java.lang.CharSequence)">setText</a>(string)        
    </code>
</pre>

```kotlin
val locale = Locale.forLanguageTag("nl-NL")
val localeSpan = LocaleSpan(locale)

val string = SpannableString("Stichting Appt")
string.setSpan(localeSpan, 0, string.length-1, Spanned.SPAN_INCLUSIVE_INCLUSIVE)

element.setText(string)
```
