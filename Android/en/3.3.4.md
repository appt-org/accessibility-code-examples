# Success Criterion 3.3.4 Error Prevention (Legal, Financial, Data)

On Android, you can use a [Dialog](https://developer.android.com/guide/topics/ui/dialogs) to confirm dispatches. Usually [AlertDialog](https://developer.android.com/reference/androidx/appcompat/app/AlertDialog) is the clearest option. But you can also use a [BottomSheetDialog](https://developer.android.com/reference/com/google/android/material/bottomsheet/BottomSheetDialog).

<pre>
    <code class="language-kotlin">
val builder = <a href="https://developer.android.com/reference/android/app/AlertDialog.Builder">AlertDialog.Builder</a>(this)
builder.<a href="https://developer.android.com/reference/android/app/AlertDialog.Builder#setTitle(java.lang.CharSequence)">setTitle</a>("Confirm Appt membership")
builder.<a href="https://developer.android.com/reference/android/app/AlertDialog.Builder#setMessage(java.lang.CharSequence)">setMessage</a>("Your bank account will be billed.")

builder.<a href="https://developer.android.com/reference/android/app/AlertDialog.Builder#setPositiveButton(java.lang.CharSequence,%20android.content.DialogInterface.OnClickListener)">setPositiveButton</a>("Proceed") { dialog, which ->
  // Proceed
}

  // Cancel
}

builder.<a href="https://developer.android.com/reference/android/app/AlertDialog.Builder#show()">show</a>()
