# Success Criterion 2.4.2 Page Titled

On Android, we recommend providing as many screens as possible with a [Toolbar](https://developer.android.com/reference/androidx/appcompat/widget/Toolbar).

<pre>
    <code class="language-kotlin">
override fun <a href="https://developer.android.com/reference/android/app/Activity#onCreate(android.os.Bundle)">onCreate</a>(savedInstanceState: Bundle?) {
    super.onCreate(savedInstanceState)
    <a href="https://developer.android.com/reference/android/app/Activity#setContentView(int)">setContentView</a>(R.layout.appt)

    val toolbar = <a href="https://developer.android.com/reference/android/view/View#findViewById(int)">findViewById</a>(R.id.toolbar)
    <a href="https://developer.android.com/reference/androidx/appcompat/app/AppCompatActivity#setSupportActionBar(androidx.appcompat.widget.Toolbar)">setSupportActionBar</a>(toolbar)
    title = "Title"
}        
    </code>
</pre>

```kotlin
override fun onCreate(savedInstanceState: Bundle?) {
    super.onCreate(savedInstanceState)
    setContentView(R.layout.appt)

    val toolbar = findViewById(R.id.toolbar)
    setSupportActionBar(toolbar)
    title = "Title"
}
```
