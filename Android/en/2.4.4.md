# Success Criterion 2.4.4 - Link Purpose (In Context)

On Android, you can use [URLSpan](https://developer.android.com/reference/android/text/style/URLSpan.html). The [ViewCompat.addLinks](ps://developer.android.com/reference/android/text/util/Linkify#addLinks(android.text.Spannable,%20int)) method can also be useful. The [ViewCompat.enableAccessibleClickableSpanSupport](https://developer.android.com/reference/androidx/core/view/ViewCompat#enableAccessibleClickableSpanSupport(android.view.View)) method is necessary to support lower versions of Android.

<pre>
  <code class="language-kotlin">
val textView = <a href="https://developer.android.com/reference/android/widget/TextView">TextView</a>(this)

val url = "https://appt.nl"
val link = "Appt"

val spannableString = <a href="https://developer.android.com/reference/android/text/SpannableString">SpannableString</a>("Learn more about $link.")

val index = spannableString.<a href="https://kotlinlang.org/api/latest/jvm/stdlib/kotlin.text/index-of.html">indexOf</a>(link)
spannableString.<a href="https://developer.android.com/reference/kotlin/android/text/Spannable#setspan">setSpan</a>(<a href="https://developer.android.com/reference/android/text/style/URLSpan">URLSpan</a>(url), index, index + link.length, <a href="https://developer.android.com/reference/android/text/Spanned#SPAN_EXCLUSIVE_EXCLUSIVE">Spanned.SPAN_EXCLUSIVE_EXCLUSIVE</a>)

textView.text = spannableString
textView.movementMethod = <a href="https://developer.android.com/reference/android/text/method/LinkMovementMethod">LinkMovementMethod</a>.<a href="https://developer.android.com/reference/android/text/method/LinkMovementMethod#getInstance()">getInstance()</a>

<a href="https://developer.android.com/reference/androidx/core/view/ViewCompat">ViewCompat</a>.<a href="https://developer.android.com/reference/androidx/core/view/ViewCompat#enableAccessibleClickableSpanSupport(android.view.View)">enableAccessibleClickableSpanSupport</a>(textView)
  </code>
</pre>

```kotlin
val textView = TextView(this)

val url = "https://appt.nl"
val link = "Appt"

val spannableString = SpannableString("Learn more about $link.")

val index = spannableString.indexOf(link)
spannableString.setSpan(URLSpan(url), index, index + link.length, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE)

textView.text = spannableString
textView.movementMethod = LinkMovementMethod.getInstance()

ViewCompat.enableAccessibleClickableSpanSupport(textView)
```
